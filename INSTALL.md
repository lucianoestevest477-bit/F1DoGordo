# Install F1 do Gordo in FL Studio

This guide is for the packaged **F1 do Gordo** VST3 release.

## Steps

1. Download the release ZIP from GitHub.
2. Extract the ZIP.
3. Find the folder named:

```text
F1 do Gordo.vst3
```

4. Copy the whole `F1 do Gordo.vst3` folder to:

```text
C:\Program Files\Common Files\VST3
```

5. Open FL Studio.
6. Go to `Options > Manage plugins`.
7. Click `Find plugins`.
8. Load `F1 do Gordo` on a Mixer Insert.

## Notes

- Copy the full `.vst3` folder, not only the file inside it.
- Windows may ask for administrator permission when copying to `Program Files`.
- This is a development checkpoint, not a final public release.
