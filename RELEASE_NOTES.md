# Release Notes

## v0.1.0-wheel-prototype

This checkpoint saves the current Wheel UI prototype as a stable development build for testing in FL Studio.

## What Works

- INPUT.
- OUTPUT.
- Compressor: THRESHOLD, RATIO, ATTACK, RELEASE, GAIN.
- AIR: MID AIR, HIGH AIR.
- DELAY: RETURN/LEVEL and TIMING.
- REVERB: ON/OFF and MIX.
- BYPASS.

## Experimental

- A/B.
- SAVE.
- Reverb is improved, but still under refinement.
- UI is based on a visual prototype and is not final public-release artwork.

## Next Phases

- Improve Delay with analog/tape-inspired behavior.
- Improve Compressor feel and gain staging.
- Improve Air tone and control range.
- Replace reference/prototype artwork with original final artwork before public release.
- Improve installer and distribution workflow.

## Build

Built as a Windows VST3 Release target through CMake and Visual Studio 2022.
